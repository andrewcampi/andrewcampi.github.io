how to create your own web page

Personal Website

Goal: 

Develop, deploy, and host a user-friendly website showcasing completed programming projects without spending any money.

The Process:

#1. Develop HTML, CSS, and JavaScript files to display each webpage

#2. Deploy the files to the host server.

#3. Connect a domain name so that users can receive the web pages when requested.

There are many website builders that will conveniently do all three of these steps, but convenience comes at a price. For this project, I must find the most convenient, free solution.

Website Building Software:
The most popular website building software is WordPress. WordPress helps developers create the website, seamlessly deploy it to their servers to be hosted, and connect a domain name. Some other similar softwares include Wix and Squarespace. While these solutions make creating a website relatively simple, they charge monthly fees.
The software that I used to create this website is Mobirise, which only completes step #1 of the process.
Github:

Github is a very popular software that will host a user’s files for others to view, edit, and download. Many programmers use it to display or collaborate on programming projects that they have completed in collections called repositories. 

Since many users of Github wanted a more user-friendly way to display and explain their repositories, Github created Github Pages. Using Github Pages, a repository of HTML, CSS, and JavaScript files for a website can be interpreted as an actual website. In addition to this, it creates a custom URL for each user’s website in the format username.github.io . Since my github username is “andrewcampi”, my Github Page URL is andrewcampi.github.io . By using Github Pages, I can upload my developed website to Github’s servers for them to host, all without spending any money.
