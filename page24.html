Calculating Digits of Pi

What is Pi:

Pi is the ratio of a circle's circumference and its diameter. This is an irrational number, meaning that pi’s value has an infinite sequence of non-repeating digits. Therefore, pi’s exact value can only be estimated. 

Approximating Pi Using Polygons:

The Ancient Greek mathematician Archimedes created the first method to approximate pi by using hexagons to determine the circumference of the circle before dividing it by the circle’s diameter. To get a more accurate approximation, he used 96 sided polygons. Archimedes calculated the value of pi to be between 223/71 and 22/7. The fraction 22/7 remains to be one of the most popular approximations of pi.
Around 600 years later, the Chinese mathematician Zu Chongzhi used the same method as Archimedes except with a 12,288 sided polygon instead of a 96 sided polygon to increase accuracy. He approximated that the value of pi is 355/113, which is correct to six decimal places.

Approximating Pi Using Computers:
To determine the value of the infinite series, an infinite amount of iterations of the series need to be calculated. Computers are exponentially faster than humans at performing tedious calculations. In 2014, a computer calculated pi correctly up to 13,300,000,000,000 decimal places.
