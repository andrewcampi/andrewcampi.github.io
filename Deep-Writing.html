Deep Writing

About Computer Generated Poetry:
Computer Poetry is a version of Deep Learning called Deep Writing. There is a lot of theory and details about Recurrent Neural Networks and Deep Learning Artificial Intelligence, but I will keep it basic here.

Deep Writing is the process by which artificial intelligence learns to create unique writing from sampled writing. With the assistance of some premade code downloaded from Github and the use of Google’s Deep Learning software called TensorFlow, your own computer will learn the language, vocabulary, and grammar of whatever you desire, and after some time has passed, it will be able to output its own unique writing. 

Use the links here to learn more about Deep Learning, or Recurrent Neural Networks, or Artificial Neural Networks, or TensorFlow.

Before we get started, you should know that this step-by-step tutorial will only work on MacOS. Windows and other OS users can teach their computer to write poetry in this fashion, but will also require some outside knowledge on how to translate Terminal (Mac) commands to something such as Command Prompt (Windows). I have seen other tutorials that use the Anaconda software to install TensorFlow in a different way, at this link. 

Steps

1


Download “word-rnn-tensorflow-master”

The code we will be using today was written by Sung Kim, a Computer Science teacher at HKUST. He has very generously posted his creation on GitHub, which is a website that host other coded creations for downloading like this one. His creation is open source, so feel free to do with it as you please, but please give credit were credit is due.


Visiting this link, locate the Green “Clone or download” button to the right. When it is clicked on, a popup will display the file name, revealing a blue “show on desktop” button and a blue “download ZIP” button. Click the “download ZIP” button to download this package called “word-rnn-tensorflow-master”. Next, locate the ZIP file in your download folder. Drag the ZIP file onto your desktop and open it.


When the blue folder titled “word-rrn-tensorflow-master” is visible on your desktop, and you are able to open the folder to reveal some files ending in the extension “.py” along with various other subfolders, you are ready for the next step.  

2


Download "Atom"

If you already installed Atom on your mac, continue on to Step #3. If you don’t know what this software is or where to install it, continue reading this step.

Atom is a software that, in very basic terms, allows the user to read and edit files that mac or windows would not allow them to read/edit, such as Python files.

To download Atom, visit this link, and click the yellow download button. This download might include some very basic installation steps, but all the steps should be labeled and explained well by the Atom Installer.

Once the Atom is installed, check by locating Atom in your applications folder, or by using Spotlight Search by clicking the Command Key and Spacebar, typing “Atom”, and clicking the Enter key. If Atom successfully opens, continue on to Step #3. 

3


Install TensorFlow

TensorFlow can be installed in various different ways with different command prompts and software, such as the Anaconda software. But to make this step as simple as possible, we will be using Mac’s Terminal Application, which is already available to use on your computer.

First, open Terminal by locating it in the applications folder, or by pressing the Command Key and Spacebar to open the Spotlight Search, and typing “Terminal” in and pressing the Enter key. Here is a link of what the Terminal icon looks like, in case you don’t know already.

When Terminal is opened, you can press the Command Key and the Plus Key to make the font size larger if you wish. Now it is time to write some commands. The list below tells (in order) the words you can copy and paste into Terminal. Copy each box into Terminal, and press enter after each box has been pasted into Terminal. 

Command #1 installs Pip in a simple package using Sudo.
>     sudo easy_install pip

Command #2 has pip install TensorFlow version 1.1.0rc0 with user permissions enabled.
>     pip install tensorflow==1.1.0rc0 --user

Command #3 checks the tensorflow version. It should say “verison: Tensorflow 1.1.0rc0”
>    pip show tensorflow

Problems are very common in this step, but there are a lot of different methods to install this program. If there is any Red Errors it might be ok, but if there are Red Errors on the last line of code in the line above where you can type, something went wrong. Command #3 will determine if you have completed this step correctly. If it is not installed correctly, try and troubleshoot the error by copying the Red Error message, and searching it into Google or another Search Engine of your liking. 

4


Insert and Manage The Sample Text

First, create a separate document (like in Pages, Word, or Google Docs) of all the writing you will use in this session to train the computer. For example, I made a document with all of Shakespeare’s 150+ sonnets. The more words, the better trained your computer will be.

Next, open the folder we installed earlier titled “word-rnn-tensorflow-master”. Open the subfolder titled “data”. Open another subfolder inside “data” titled “tinyshakespeare”. Open the text file titled “input.txt”.

Now, copy everything from your document you just made, and delete all the words in the “input.txt” file, and paste the content of your document in there. Once you see that the document’s content have successfully be inserted into “input.txt”, close out of “input.txt”

In the “word-rnn-tensorflow-master” folder”, locate the file titled “sample.py”. The mac software prevents you from opening it. This is why we installed Atom earlier.

Open Atom by locating it in your applications folder or by using Spotlight Search. Once Atom is opened, carefully drag “sample.py” into the middle section of the Atom application. If you do this successfully, you should see “sample.py” opened and be able to read the code.
Scroll down slightly until you see this line of code: 
>    parser.add_argument(‘-n’, type=int, default=200, help=’number of words to sample’)

This line of code determines the number of words that the computer will output after it has learned your sample text. The computer is defaulted to output only 200 words. I recommend this number to be between 1000 and 1500 for the highest quality.
Change the default=200 to default=1500 so that it looks like this:
>    parser.add_argument(‘-n’, type=int, default=1500, help=’number of words to sample’)

5


Train The Model

It’s time to reopen Terminal and write some commands like before, only these will take some time for the computer to process. Make sure your computer is charging because these computations are intensive for the computer and will drain the battery life rather quickly. The higher quality the computer, the faster the computations will be.

Just like before, copy the command in the box and paste it into Terminal. Make sure to press enter after each individual box has been pasted into Terminal. 

Command #1 has the system look in the requested folder.
>      cd ~/desktop/word-rnn-tensorflow-master

Command #2 has Python run the file “tran.py”. This file will start the training process of Deep Writing, where the computer reads and learns what is in the file titled “sample.txt”:
>     python train.py

If the cursor is blinking on a line below all other writing, that means the computer is thinking. Do not close out of Terminal or type anything in Terminal while it does this. On my 2017 Macbook Pro, this Command takes about 5 to 10 minutes to completely process. If your document content is very big, for example all the works of Shakespeare rather than only his sonnets, this command might take a few hours to completely process. The longer the computer processes information, the better the outcome will be. The process will be completed when it says it has saved its progress and has finished leaning. 

6


Outputting Unique Poetry

Now that the computer has learned the sample text you provided, it is now time to see what it can do with its new knowledge of poetry.
We will now write some more commands in Terminal using the same
methods of copying what is in the box and pressing enter after each individual command has been pasted into Terminal.

Command #1 has Python run the file “sample.py”. This file will cause the computer to output the unique poetry.
>      Python sample.py

It may take a couple minutes, but eventually the computer will output 1500 words of poetry written in Deep Writing. After a little bit of formatting and slight polishing, its ready to been shown to the world.  

Example Output (Trained on Shakespeare's Sonnets):

SONNET XXXVII
And is thy old ears see fame yet,
hoisted state from day of my pen, play'st woe;
for Time's predict in here's doth, thou simple ear,
why consecrate hath expired knights,
and still to die with that fill'd prove; O thee, my Time's skill.
That which thy love, despite what thy pilgrimage did doth 'No.'
To speak as my assistance in my Fortune say, or on my cross,
o prize to fear thy beauty's strong jewels.
Though conscience is thee, but thy thought is so sweet,
whereon you, I rank and mayst doth qualify.
Then yet a sinful right thy very hate, towers, thou most, the heart's eye.
Moon of all wealth, remember'd blunt absence to thee, Ah! Thy sense.
Thou, oft a more daily dost the darling spent, and shame!
Permit some love and be not in shame; stopped face, crush'd by untruth.
