Vulernable Machines 

This is a collection of custom, intentionally vulnerable machines created by me. To practice hacking and for educational purposes, I have created these machines with specific vulnerabilities and security oversights. These machines can be downloaded, installed, and practiced upon. I have also included a writeup of how each machine is intended to be hacked. I have written these writeups using a template provided by Hack The Box, an online hacking sandbox.
